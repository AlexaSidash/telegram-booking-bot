# 🤖 Telegram Booking Bot

Telegram bot + website for booking services with notifications.

## ✨ Features
- 📅 Online booking through the website
- 🤔 Smart time selection (only available slots)
- 📱 Telegram notifications about new bookings
- 👑 Admin panel to view all reservations
- 💾 SQLite database
